## Getting Started

1. `git clone https://github.com/ManzDev/css-futbol`
2. `cd css-futbol`
3. `sudo npm install -g parcel-bundler`
4. `git checkout 16-refactor` (or other branch)
5. `npm install`
6. `npm run dev`
